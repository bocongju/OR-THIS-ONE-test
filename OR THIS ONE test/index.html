Not only supplement

「夕方に眠くて集中できない」「ストレスなのかな？炭酸水が飲みたくて仕方がない」 なんとなくの不調でスッキリしない。それらはあなたのカラダからのサインなんです。
パフォーマンスを上げるためにはバランスのとれた食生活が必要なのは分かっているけど忙しくて。これは食べなきゃいけない、これは食べちゃいけない、って考えるだけでストレス。そんなことってありますよね。

「OR THIS ONE」がおすすめするのは２つの新しい習慣です。
一つは、自分だけのカラダからのサインに合った、自分だけにピッタリな生活改善にチャレンジすること。
もう一つは、栄養素を補う（＝Supplement）のだけではなく、カラダが喜ぶものに置き換えて（＝Replacement）食べること。

OR THIS ONE」サプリメントに使われている原料は自社基準で厳選。素性のしれない原料は使用しておりません。
野菜類はほとんどが国産無農薬のものを使用し、一部国内にないものは海外のオーガニック野菜を使用しています。天然でオーガニックであることが必ずしも良いというわけではありません。大切なは、お客様にとって最適であること。全体のバランスを考えて、天然である必要がある成分は天然原料を使用しますし、合成原料の方が良い、または天然原料を使用する必要がない場合には合成ビタミンなどの合成原料を使用します。商品によっては機械詰めによる加熱/加圧を避けるために人の手でカプセルに詰めているものもあります。

誰よりも誠実なサプリメントでありたい。
「OR THIS ONE」はそう考えています。

ずっと長く続けるために
「置き換える」

例えば、いくらカラダにいいと分かっていても「生レバーを一日に４皿食べてください」なんて無理な話です。また例えば、食が細かったりすると「毎食たんぱく質を摂ってください」ってこと自体がストレスになったりします。体質改善や生活習慣の改善は、もっと身近で楽しいものでいいと「OR THIS ONE」は思っています。
私たちのカラダは私たちが食べるものから出来ています。カラダにいいものばかり食べれればいいですけど、毎日の生活の中ではそうもいきませんよね。苦手で食べづらいものだってあります。そんな時は「OR THIS ONE」が助けてくれます。
「〇〇〇 OR THIS ONE（必要な栄養素を含む〇〇〇を食べるのか、このサプリメントを飲むのか）」カラダに必要な食べものを必要な量だけ摂取することがむずかしいときに「OR THIS ONE」を活用してください。

そんな貴方のために間食に置き換えるためのナッツを作りました。いつものおやつのかわりにカラダが喜ぶ栄養素がたっぷりのナッツを食べましょう。

成人女性には一日に約10mgのヘム鉄が必要であるとされています。これを食事から摂取しようと思うとレバ刺しを毎日４皿ずつ食べなければいけません。へむてっつ（鉄分ナッツ）には100gあたり4.97mgの鉄分が含まれています。間食にかわりに置き換えることで手軽にカラダに必要な栄養素を摂取しましょう。
